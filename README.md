Stock Portfolio Manager | CS50 Finance
📌 Project Overview
A dynamic web application that allows users to manage a simulated investment portfolio. Users can "buy" and "sell" stocks in real-time using the IEX API, track their transaction history, and view their current balance.
+1

🛠️ Tech Stack
Backend: Python & Flask 
+1

Database: SQL (SQLite) for user data and transaction history 
+1

Frontend: HTML5, CSS3, Bootstrap

API: Integrated IEX Cloud API for real-time market data

🚀 Key Features
Real-time Quotes: Instant access to current stock prices.

Portfolio Management: Automatic calculation of total asset value plus cash balance.

Transaction Logs: Detailed history of all trades performed by the user.

Security: Password hashing and session management.

⚙️ Installation & Setup
Clone the repository.

Install dependencies: pip install -r requirements.json.

Set your API Key: export API_KEY=your_key_here.

Run the application: flask run.
